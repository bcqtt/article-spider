# article-spider 为技术实练项目
###(JAVA项目)网络爬虫-webmagic-bootstrap
项目框架由SpringMVC,Spring4,MyBatis3集成

###项目集成的内容主要有：
* 1、支持网页爬虫
* 2、多线程、线程池
* 3、支持全文搜索
* 4、支持Hadoop分布式平台、HDFS/MapReduce、Zookeeper、HBase
* 5、支持redis分布式缓存
* 6、集成微信公众号开发
* 7、Spring4新特性
* 8、ActiveMQ
* 9、Nginx详细配置
* 10、OpenCV图像处理技术
* 11、缓存Ehcache
* 12、java设计模式练习、Demo
* 13、java8练习
